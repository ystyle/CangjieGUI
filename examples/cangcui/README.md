# cangcui：苍翠画卷（千里江山 · 青绿山水）

框架同名示例：一幅约 3:1 的横卷（1200 × 400 逻辑尺寸），以千里江山图式的青绿山水呈现
「苍翠」之名——远山如黛、江水穿行于近处的岛群之间，主峰立于黄金分割、松冠其上，
乌篷小舟横于开阔江面，引首竖排题字与朱印。整幅画面由渲染原语程序化绘出，同尺寸永远
同画（快照可回归），窗口拉伸时山川随宽高连续重排。

## 实现设计

- **山形有聚有散**：山脊线 = 三组正弦叠加经「峰锐谷缓」幂次整形，再乘一条沿卷轴起伏的
  聚散包络——群峰之间留出低谷；五重山各用不同的包络频相与锐度（远山圆浑、近山峭拔），
  近两重再加高斯主峰（近景主峰落在黄金分割附近），主宾之势分明
- **江水穿山**：水线横贯 0.80 画高；远三重山立于水后，近两重山的竖条只画到水线——山脊
  沉入水下的横段不落一笔，山群成岛、岸线随山势弯转，江水由此穿过群山；岛身之下随形
  压一抹淡倒影
- **设色勾线**：逐 2px 竖条「石青罩顶 → 石绿铺底」分染（`fillRoundedRectGradient`），
  沿脊短段折线勾勒（近层墨青、远层淡描，入水即止）——青绿山水「勾线填色」技法的直译
- **水法**：网巾纹式的下弯涟漪弧 + S 形流水长曲，写「潺潺流水」的走势；
  `fillRoundedRectSoft` 的羽化雾带横锁远山脚与江腰
- **点景**：中国画式的松（虬曲三折主干 + 三层平顶针冠 + 旁逸侧枝，一正一欹成簇，只植于
  高出水线的山脊）、乌篷小舟（月牙船身、竹篷拱起、渔翁执篙，只落在两重近山都沉于水下的
  开阔江面）、天际人字飞鸟——位置出自候选表，是否成景由山水关系动态判定
- **引首**：竖排「苍翠」大字与「千里江山」旁注用系统安装的霞鹜文楷（`Fonts.register` 个人
  字体目录的 `LXGWWenKai-Regular.ttf`，缺失时自动回退平台 UI 字体）；朱文小印的印文按所用
  字体实测宽高在印面内取中（含光学下移补正）
- **静态零开销**：控件不请求帧、不吞事件，首帧绘出后渲染循环即回到脏帧跳过的空闲

## 文件结构

| 文件 | 职责 |
|---|---|
| [main.cj](src/main.cj) | 入口；3:1 横卷窗口 |
| [scenery.cj](src/scenery.cj) | 山形函数、层次色板、点景位置表（纯常量与纯函数） |
| [painting.cj](src/painting.cj) | `ScrollPainting` 控件：按景深次序绘出整幅画 |
| [views.cj](src/views.cj) | 整窗即画 |
| [theme.cj](src/theme.cj) | 绢本底色主题 |

## 运行

```powershell
cd examples/cangcui
cjpm run
```

支持视觉回归快照（本仓库画廊封面即由此生成）：

```powershell
cjpm run --run-args "--snapshot cangcui.bmp"
```
